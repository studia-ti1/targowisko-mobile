# targowisko - flutter app
    
## 1. Basic Commands
- `flutter analyze` - linting
- `flutter run` - start app
- `flutter format .` - format all files in current directory
- `flutter pub global run devtools` (or `devtools`) - open dev tools
  
## 2. Identifiers styles  
- **`PascalCase`** - classes, enums, typedefs, and type parameters   
- **`snake_case`** - libraries, packages, directories, source files and import prefixes
- **`camelCase`** - variables, constants (rest)
